一. 如何使用?

（1）直接下载方式:

将verless.less下载到本地，然后在自己的样式文件中引入，如下：

@import 'veryless.less';

（2）Node方式: 不熟悉node，可以不使用此方式

首先，确认你的机器已经安装node与npm

npm install -g veryless

mkdir test

veryless -init test #初始化完成并下载了veryless

veryless -use css3 test #下载了css3插件

二. 小试牛刀

（1）CSS3:

动画 (Demo)：

.animation-name: shake;

阴影 (Demo)：

.box-shadow(5px 5px 10px #ccc);

透明：

.opacity(80);

圆角 (Demo)：

.border-radius(10px);

旋转 (Demo)：

.rotate(10deg);

（2）Grid (Demo):

创建24格栅格布局：

.makeGrid();

创建24格流式布局：

.makeFluid();

创建6格Metro布局：

.makeMetro();
